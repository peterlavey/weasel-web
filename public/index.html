Preparando ambiente

Primero necesitamos tener nodejs en nuestro equipo, puedes descargarlo en este link. Ingresamos a la carpeta de weasel, abrimos la terminal y ponemos npm install && npm start Finalmente de debe modificar el archivo proxy de nuestra aplicación para que apunte a los servicios de localhost

Paso 1

Crear el directorios, se recomienda la siguiente: Proyecto -> vista -> prueba esa prueba puede ser un caso de error, etc..

Paso 2

Crear los servicios necesarios dentro del directorio, se guardará en el menú de busqueda y quedará disponible en el dictorio actual.

Paso 3

Iniciar, se levantarán todos los servicios del directorio actual, se puede cambiar puerto en el botón con el engranaje.

Menú de busqueda

Aquí es donde se alojan todos los servicios creados, por lo tanto aquí es donde se eliman de forma definitiva, también permite buscar y agregar un servicio al directorio actual haciendo click sobre él.
